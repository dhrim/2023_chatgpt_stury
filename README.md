# 방법

- 스터디 참석 인원이 자료의 cookbook의 노트북에서 하나를 골라 스터디 전에 실행 시켜보고 파악한다.
- 스터디에서 준비한 노트북을 설명한다.
- 1회에 4명, 1주 1번
- OT 1번 + 스터디 7번
- OT 때에는 임도형이 ChatGPT에 대하여 설명하고, 스터디에 필요한 환경을 설명. 그리고 스터디 방법을 설명한다.
